### Основные понятия:
- **База данных** — это структурированный набор данных, представленный в виде таблиц.
- **Таблица** состоит из строк (кортежей) и столбцов (атрибутов), отображающих данные об объектах реального мира.

### A
- AND - логическая операция "**И**" или **конъюнкция**, означает, что все условия должны возвращать **true**, в противном случае результат будет **false**, если хотя бы одно из условий будет **false**;

